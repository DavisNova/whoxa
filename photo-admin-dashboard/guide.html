🚀 快速开始

启动本地服务器

双击 start-server.bat 文件，或在浏览器中直接打开 index.html

配置 API 连接

进入"系统设置"页面，修改 API 地址为您的服务器地址

开始管理照片

切换到"照片管理"页面，查看和管理上传的照片

⚙️ API 集成

当前系统使用模拟数据进行演示。要连接到真实的 Beimo API，请修改以下配置：
